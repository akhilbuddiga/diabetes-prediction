# Diabetes Prediction Project

## Project Description
This project aims to predict the likelihood of diabetes in patients based on health parameters such as glucose levels, BMI, age, blood pressure, and other features. It involves data cleaning, visualization, model training, and evaluation using Python and machine learning techniques.

## Steps
1. Load raw data
2. Clean data
3. Explore data
4. Visualizations
5. Prepare and train model
6. Evaluate model
